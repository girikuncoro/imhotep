# Variadic Functions (one more thing...)

## TLDR;

* Variadic functions are used thru out GO packages. 
* Function can take 0 or more arguments of a given type.

```go
func ak47(args ...int) {
	fmt.Println(args)
}

func main() {
	ak47()
	ak47(1)
	ak47(2, 3, 4)

	s := []int{1, 2, 3}
	ak47(s...)
}
```
